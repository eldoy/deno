### Commands

```
# Run program
deno run
deno run --allow-net
deno run --allow-net='eldoy.com'
deno run --allow-net --watch src/index.ts

# Lint, set rules in deno.jsonc
deno lint

# Format
deno fmt

# Check typescript
deno check
```
